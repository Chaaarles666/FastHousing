# FastHousing — Cursor 开发 Prompt

## 使用说明

以下 Prompt 按开发顺序排列。每个 Prompt 独立可执行，在 Cursor 中逐个使用。
建议先用 Prompt 0 初始化项目，然后按 1→2→3 顺序开发各模块。

## Prompt 0: 项目初始化

```

## Prompt 2: 交易 Checklist

```
在 FastHousing 项目中开发"交易流程 Checklist"页面 (/checklist)。

## 核心设计

整个购房流程分 7 个阶段，每个阶段是一个可折叠的 Accordion 面板。

## 数据结构

interface ChecklistPhase {
  id: string;
  title: string;
  icon: string;           // emoji
  estimatedDays: string;  // 预计耗时
  items: ChecklistItem[];
  pitfalls: string[];     // 常见坑点
  documents: string[];    // 需准备的材料
  tips: string[];         // 实用建议
}

interface ChecklistItem {
  id: string;
  text: string;
  checked: boolean;
  subItems?: string[];    // 子项说明
}

## 7个阶段的完整内容

### 阶段1: 购房准备 🏁
预计耗时：1-2周
待办：
- 确认购房资格（上海限购政策自查）
- 拉取个人征信报告（央行征信中心，免费每年2次）
- 查询公积金账户余额和月缴存额
- 评估家庭财务状况（可跳转购房能力评估页）
- 确定预算范围和目标区域
- 了解目标区域近期成交行情
- 列出购房核心需求清单（必须有vs最好有）

坑点：
- ⚠️ 社保/个税断缴可能影响购房资格，提前确认
- ⚠️ 征信有逾期记录会影响贷款，至少提前半年清理
- ⚠️ 不要提前大额消费或新增贷款，影响贷款审批

材料：身份证、户口本、婚姻证明、社保/个税缴纳证明、征信报告、公积金缴存证明

### 阶段2: 看房选房 🔍
预计耗时：2-8周
待办：
- 制定看房路线和计划
- 实地看房至少15-30套（别急着决定）
- 每套房记录关键信息（建议用房源对比器）
- 同一套房不同时段各去一次（白天看采光、晚上看噪音、雨天看渗水）
- 和小区保安/业主聊聊实际居住感受
- 查看小区物业费、停车费标准
- 了解小区近期有无重大规划（拆迁、修路、建学校等）

坑点：
- ⚠️ 中介带看的都是"好角度"，要自己仔细转一圈
- ⚠️ "满五唯一"直接影响几十万税费，务必核实
- ⚠️ 顶楼漏水、一楼潮湿、腰线层遮光——这些问题看照片看不出来
- ⚠️ 距离地铁"步行5分钟"可能实际要15分钟，自己走一遍
- ⚠️ 检查房屋是否有"凶宅"历史（可以问邻居、查法院公告）

材料：手机（拍照录像）、卷尺、指南针APP、看房笔记模板

### 阶段3: 谈价签约 💰
预计耗时：1-2周
待办：
- 查询同小区近3-6个月成交价（贝壳/链家成交记录）
- 了解卖家售房原因和紧迫程度
- 制定议价策略（建议在成交均价基础上先压5-10%）
- 审查房屋产权状态：
  - 有无抵押？
  - 有无查封？
  - 是否有共有权人？共有权人是否同意出售？
  - 是否有租约？租客是否放弃优先购买权？
- 签署购房合同，重点审核：
  - 付款方式和时间节点
  - 违约责任和赔偿比例
  - 交房时间和条件
  - 户口迁出期限
  - 附属设施（家具家电）清单
- 支付定金（建议不超过总价5%，最多不超过20%）

坑点：
- ⚠️ "定金"受法律保护不可退，"订金"可以退——合同上用哪个字要看清楚
- ⚠️ 合同必须写明"如贷款审批未通过，定金全额退还"
- ⚠️ 要求合同注明"卖方保证该房屋非凶宅，无重大瑕疵，否则双倍返还定金"
- ⚠️ 阴阳合同有法律风险，且现在税务系统核验价对比严格
- ⚠️ 确认卖方是产权人本人签字，代理人需有公证委托书

材料：购房合同（建议请律师审核）、身份证、定金（银行转账留凭证）

### 阶段4: 贷款申请 🏦
预计耗时：2-4周
待办：
- 选择贷款方式：纯商贷 / 纯公积金 / 组合贷
- 对比至少3家银行的利率和放款速度
- 选择还款方式：等额本息（月供固定）vs 等额本金（总利息少）
- 准备贷款申请材料
- 提交贷款申请
- 等待银行审批
- 签署贷款合同

坑点：
- ⚠️ 申请贷款前3-6个月不要有大额消费贷、新信用卡申请
- ⚠️ 银行流水不够？提前6个月做准备，或补充其他资产证明
- ⚠️ 公积金贷款审批慢（4-8周），卖方可能不接受，提前沟通
- ⚠️ 组合贷需要同一家银行办理商贷部分
- ⚠️ 贷款金额可能不如预期，提前有 Plan B

材料：收入证明（公司盖章）、银行流水（6个月）、征信报告、购房合同、身份证/户口本/婚姻证明、公积金证明

### 阶段5: 过户缴税 📝
预计耗时：1-2周
待办：
- 计算并确认各项税费金额
- 到不动产交易中心办理过户
- 缴纳契税、增值税、个人所得税
- 领取新不动产权证（通常5-10个工作日）
- 确认卖方户口已迁出

坑点：
- ⚠️ 税费由谁承担（买家or卖家）要在合同里写清楚
- ⚠️ 注意"核验价"——税务系统有最低计税价，不一定等于成交价
- ⚠️ 过户前再次确认房屋无新增抵押/查封
- ⚠️ 不动产权证上的名字、面积要核对无误

材料：双方身份证原件、不动产权证原件、购房合同、完税证明、银行贷款合同

### 阶段6: 交房验收 🔑
预计耗时：1-3天
待办：
- 验房清单逐项检查：
  - 水电燃气是否正常通畅
  - 墙面地面天花板有无裂缝、渗水、发霉
  - 所有门窗开关是否正常，密封条有无老化
  - 厨房卫生间下水是否通畅
  - 房屋面积与合同是否一致
  - 合同约定的家具家电是否齐全完好
- 抄表记录：水表、电表、燃气表当前读数
- 确认卖方已结清水电燃气物业费
- 收取所有钥匙（大门、单元门、信箱等）
- 拍照或录像留档

坑点：
- ⚠️ 验房一定要在付清尾款之前
- ⚠️ 问清物业有无欠费，有的小区欠费会锁门禁卡
- ⚠️ 检查是否有违建（如封阳台、打通墙体），后续可能被要求拆除

材料：验房工具（卷尺、水平仪、手电筒、乒乓球测地面水平）

### 阶段7: 入住善后 🎉
预计耗时：持续
待办：
- 办理物业交接手续
- 更换大门锁芯（安全第一）
- 开通/过户水电燃气网络
- 办理户口迁入（如需）
- 办理不动产权证加名（如需）
- 通知银行更新通讯地址
- 规划装修（如需）
- 认识邻居，加入业主群

坑点：
- ⚠️ 换锁是第一件事，别犹豫
- ⚠️ 装修前查看小区装修规定（时间、噪音、建材搬运）
- ⚠️ 保留好所有购房材料原件，至少保存到贷款结清后

材料：不动产权证、身份证、户口本

## 页面设计

1. **顶部**：总体进度条（百分比 + 当前阶段提示）
2. **7个折叠面板**（Accordion），每个面板包含：
   - 标题行：阶段名 + emoji + 预计耗时 + 完成状态标签
   - 展开内容：
     - 待办列表（Checkbox，可勾选，状态持久化到 localStorage）
     - ⚠️ 避坑指南（橙色/红色卡片）
     - 📋 材料清单（可折叠子区域）
3. **底部浮动**：当前阶段的下一步建议

进度数据存储 localStorage，key 为 "fasthousing_checklist"。

每个 checkbox 勾选后有轻微动画反馈。阶段全部完成时显示 ✅ 标记。

注意：内容很长，每个阶段默认折叠，只展开当前进行中的阶段。
```

## Prompt 3: 购房能力评估 & 财务规划

```
在 FastHousing 项目中开发"购房能力评估"页面 (/calculator)。

这不是一个普通的房贷计算器。它是一个家庭财务规划工具，帮助用户回答："我到底该买多少钱的房子？"

## 数据模型

interface FinancialProfile {
  // 收入
  monthlyIncome: number;        // 本人月收入（税后）
  spouseIncome: number;         // 配偶月收入（税后）
  otherIncome: number;          // 其他月收入（租金、理财等）
  
  // 支出
  monthlyExpense: number;       // 月固定生活支出
  
  // 资产
  savings: number;              // 可动用存款（万元）
  providentBalance: number;     // 公积金账户余额（万元）
  providentMonthly: number;     // 公积金月缴存额（个人+单位）
  
  // 负债
  existingMortgage: number;     // 现有房贷月供
  carLoan: number;              // 车贷月供
  otherDebt: number;            // 其他贷款月供
  
  // 购房条件
  isFirstHome: boolean;         // 是否首套
  familySize: number;           // 家庭成员数
  targetArea: number;           // 目标面积（用于税费计算）
  loanYears: number;            // 贷款年限（默认30年）
}

## 页面分为4个区域

## 计算逻辑（核心）

### 等额本息月供
M = P × [r(1+r)^n] / [(1+r)^n - 1]
P=贷款总额, r=月利率(年利率/12), n=总月数(年数×12)

### 等额本金月供
第k个月月供 = P/n + (P - P×(k-1)/n) × r
P=贷款总额, n=总月数, r=月利率

### 最大可承受总价（按月供收入比反推）
maxMonthlyPayment = (totalIncome - totalExpense - existingDebt) × ratio
maxLoan = 反推贷款额（已知月供、利率、年限）
maxTotal = maxLoan / (1 - downPaymentRate) 

### 公积金最高贷款额
取 min(公积金政策上限, 按还款能力计算的额度, 按房价计算的额度)
