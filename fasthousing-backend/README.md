# FastHousing Backend

FastAPI + PostgreSQL + Redis backend for FastHousing.

## Quick Start

1. Copy env:

```bash
cp .env.example .env
```

2. Start services:

```bash
docker compose up --build
```

3. Open docs:

- http://localhost:8000/docs

## API Modules

- `/api/communities`
- `/api/listings`
- `/api/transactions`
- `/api/analysis`
